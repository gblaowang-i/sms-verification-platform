# 🚀 部署目录

本目录包含Linux服务器部署相关的所有文件和配置。

## 📁 文件说明

### 📋 部署文档
- **LINUX-DEPLOY.md** - Linux服务器详细部署指南
- **README.md** - 本说明文件

### 🔧 部署脚本
- **deploy-linux.sh** - Linux服务器一键部署脚本

### ⚙️ 配置文件
- **env.production** - 生产环境配置文件
- **server.js** - Express服务器文件（用于生产环境）

## 🚀 快速部署

1. **给脚本执行权限**
   ```bash
   chmod +x deploy-linux.sh
   ```

2. **运行部署脚本**
   ```bash
   ./deploy-linux.sh
   ```
